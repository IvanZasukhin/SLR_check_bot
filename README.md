# SLR Check Bot

Анализатор трафика игровой сессии на порту **3040** для определения участников (Players vs Bots).

## Возможности

- 🔍 **Мониторинг в реальном времени** — перехват TCP/UDP трафика на порту 3040
- 📦 **Deep Protobuf парсинг** — рекурсивный обход вложенных сообщений
- 🤖 **Автоматическое определение** — боты (`AI_*`) и игроки (`steam_*` / `device_*`)
- 📊 **GUI таблица** — имя, идентификатор, тип участника
- 💾 **Экспорт** — сохранение в CSV или Excel
- 🖱 **Клик по боту** → ник копируется в буфер обмена
- 🌐 **Двойной клик по игроку** → открывается Steam профиль

## Установка

### Способ 1: Из исходников

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/YOUR_USERNAME/slr-check-bot.git
cd slr-check-bot

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Запустите
python main.py
```

### Способ 2: Готовый EXE

1. Скачайте релиз со страницы [Releases](https://github.com/YOUR_USERNAME/slr-check-bot/releases)
2. Распакуйте архив
3. Запустите `SLRCheckBot.exe` **от имени администратора**

## Использование

### Мониторинг в реальном времени

1. **Запустите программу** от имени администратора
2. Нажмите **"▶ Старт мониторинга"**
3. **Запустите игру** — программа перехватит пакеты на порту 3040
4. Наблюдайте за счётчиками:
   - 🤖 **БОТЫ** — количество ботов
   - 👥 **ИГРОКИ** — количество игроков
5. **Клик по боту** → ник копируется в буфер
6. **Двойной клик по игроку** → Steam профиль в браузере
7. **"⏹ Стоп"** → остановка и экспорт результатов

### Экспорт результатов

- **Экспорт CSV** — таблица в формате CSV
- **Экспорт Excel** — таблица в формате .xlsx

## Требования

- Windows 10/11
- **Npcap** — [скачать](https://npcap.com/#download) (отметить "WinPcap API-compatible Mode")
- Python 3.10+ (для запуска из исходников)
- **Административные права** (для захвата пакетов)

## Сборка в EXE

```bash
# Установка зависимостей
pip install -r requirements.txt

# Сборка
pyinstaller --clean SLRCheckBot.spec

# Или просто запустите:
build.bat
```

Готовый файл: `dist/SLRCheckBot.exe`
